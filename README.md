# java-web-api-maven-spring-micrometer-hello

## Description
Another way to serve web content.
Uses micrometer to expose endpoints
through springframework actuator.

## Tech stack
- java
- maven

## Docker stack
- openjdk:11-jdk

## To run
`sudo ./install.sh -u`
- Available http://localhost/actuator

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
